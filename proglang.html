Programming Language Terminology

Term Definition
Machine Language A program written for modern computers consisting of sequences of instructions that are encoded as numeric digits
Debugging The process of locating and correcting errors in code
Assembler A program that converts mnemonics into machine language instructions
Assembly Language A collective of assemblers
Translator Converts progrmas expressed in primatives into machine language commands
Compiler Compiles several machine language instructions into one stored command
Interpreter Similar to a compiler (see above), but it immediately executes instead of storing the command
Cross-Platform Software A program capable of being properly executed across several operating systems
Natural Language Languages spoken by humans (ex. English, Spanish, Mandarin Chinese)
Formal Language Programming Languages
Machine Independence Code written to be non-reliant on a particular machine to be executed properly
High Level Programming Language Programming language written to be more easily comprehensible by humans
Object-Oriented Programming (OOP) A part of a kernel, used to communicate with controllers to execute actions on devices connected to the computer
Scripting Language Used for simple, automated tasks, most commonly used for administrative purposes
Integer Numeric data, whole numbers (ex. 49)
Float Numeric data, possibly non-whole numbers (ex. 49.06)
String Non-numeric characters (ex. "cat")
Primitive Data Data that doesn't require much processing power to represent
Data Structure The arrangment of data in code
List In Python, this is multiple items stored in a single variable
Tuple In Python, this is an ordered set of values
Set In Python, this creates a set, unordered object
Dictionary In Python, this stores data in key:value pairs
Function When used, executes a specific action (ex. sqrt(x) returns the square root of "x")
Local Variable A variable only accessible within the function where it is defined
Global Variable A variable that can be used throughout the program
Source Program Text file containing high-level programs
Object Program A program into which another program is translated by an assembler/compiler
