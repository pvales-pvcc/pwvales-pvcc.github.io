Operating System (OS) Terminology

Term Definition
Operating System (OS) The overall system of control over a computer (ex. Windows, MacOS, Linux)
Job Queue A list of actions a computer can do waiting in storage to be executed
FIFO "First In, First Out", The ordering system for a queue, where the first actions in the queue are the first actions executed
Interactive Processing An OS that communicates (or interacts) with users via remote terminals
Time-sharing An OS capable of servicing the needs of multiple users at once
Multiprogramming A style of the time-sharing OS which divides sections of time into intervals, in which only one action can be completed in a set ammount of time
Application Software The programs for performing tasks particular to the machineâ€™s utilization
System Software Performs tasks that are common to computer systems in general
Utility Software Performs programs for performing activities that are fundamental to computer installations but not included in the operating system
Graphical User Interface (GUI) A user interface system in which functions of a computer are displayed as icons on the desktop
Shell A user interface system in which functions of a computer are executed using a keyboard
Kernel Part of a computer's OS, a kernel contains the software components that perform the very basic functions required by the computer installation
Device Driver A part of a kernel, used to communicate with controllers to execute actions on devices connected to the computer
File Manager Coordinates the usage of a computer's main memory
Virtual Memory A system in which a computer is led to believe it has more memory capacity via shuffling of pages (small packets of memory) between main memory and mass storage
Boot-strapping The system of a computer transferring the OS from mass storage to main memory every time it is turned on, or "booted" up
ROM "Read Only Memory", Non-editable data permanately written, even when a computer is powered off
Deadlock When two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another
Spooling When data is held for output until a more convenient time
Super-user (or Administrator) A user with an account that posesses much higher priveleges to an OS compared to a regular user, an admin is allowed access to edit the OS and account privileges
Multi-factor Authentication A system where an account user must not only input a password, as well as verifying their identity on another device, such as a smartphone
